# 🐯 Tiger AI — دليل الإعداد والاستخدام

تم دمج **NVIDIA NIM API** في متجر Tiger Jeans بنجاح! المساعد الذكي يعمل على كل صفحات الموقع بدون تغيير أي ميزة حالية.

---

## 📋 الملفات الجديدة (لم يتم تعديل أي ملف موجود)

| الملف | الوصف |
|------|-------|
| `css/tiger-ai.css` | تنسيقات المساعد (معزولة تماماً عن تصميم الموقع) |
| `js/tiger-ai.js` | المساعد الذكي الرئيسي (يظهر للعملاء والأدمن) |
| `js/tiger-ai-admin.js` | منطق لوحة إعدادات AI |
| `database.rules.updated.json` | قواعد Firebase المحدّثة (يجب تطبيقها) |

## ✏️ الملفات المعدّلة (تعديلات بسيطة جداً)

- **`admin.html`**: إضافة تبويب "مساعد AI" في القائمة الجانبية + القائمة المتنقلة + محتوى التبويب + 2 سكريبت
- **كل صفحات العملاء** (16 صفحة): إضافة `<script src="js/tiger-ai.js"></script>` فقط

لم يتم تغيير أي تصميم، أي ميزة، أي زر، أو أي سلوك حالي.

## 🚀 خطوات الإعداد (مرة واحدة)

### 1️⃣ احصل على مفتاح NVIDIA API

1. ادخل على: **https://build.nvidia.com/settings/integrations**
2. اضغط على **"Generate API Key"** في قسم API Keys
3. انسخ المفتاح (يبدأ بـ `nvapi-...`)

### 2️⃣ حدّث قواعد Firebase (مهم جداً للأمان)

القاعدة الحالية تسمح للجميع بقراءة `settings`. يجب تحديثها لحماية مفتاح API.

1. ادخل على: **Firebase Console → Realtime Database → Rules**
2. انسخ محتوى ملف `database.rules.updated.json` (الموجود في المشروع)
3. الصقه في محرر القواعد واضغط **Publish**

هذا يضمن أن:
- ✅ العملاء يقدرون يقرؤوا `aiConfig/public` (لمعرفة هل المساعد مفعّل)
- 🔒 الأدمن فقط يقدر يقرأ `aiConfig/secret` (المفتاح والبرومبت)

### 3️⃣ فعّل المساعد من لوحة التحكم

1. ادخل على: **https://tiger-jeans.com/admin.html**
2. سجّل دخول كالمعتاد
3. من القائمة الجانبية: **مساعد AI**
4. اضبط الإعدادات:
   - ✅ **تفعيل المساعد**
   - 🔑 **مفتاح NVIDIA API**: الصق المفتاح
   - 🤖 **النموذج النصي**: اختر (Llama 3.1 70B افتراضي — متوازن وسريع)
   - 👁️ **نموذج الرؤية**: اختر (Llama 3.2 90B Vision افتراضي)
   - 🧠 **نموذج التفكير**: اختر (DeepSeek R1 افتراضي)
   - ⚙️ **المزايا**: فعّل/اطفئ أي ميزة حسب رغبتك
5. اضغط **"اختبار الاتصال بـ NVIDIA"** للتأكد أن المفتاح يعمل
6. اضغط **"حفظ إعدادات المساعد الذكي"**

## 🛠️ استكشاف الأخطاء

### المساعد لا يظهر على الموقع
- تأكد أنك سجّل دخول كأدمن وفعّلت المساعد من لوحة التحكم
- افتح Console (F12) وتحقق من عدم وجود أخطاء

### "مفتاح NVIDIA API غير صالح"
- تأكد أن المفتاح يبدأ بـ `nvapi-`
- ادخل على https://build.nvidia.com/settings/integrations للتأكد
- استخدم زر "اختبار الاتصال" في لوحة التحكم

### "permission_denied" عند الحفظ
- يجب تحديث قواعد Firebase (انظر الخطوة 2)
- يجب أن تكون مسجل دخول كأدمن

### المساعد لا يجيب عن أسئلة المنتجات
- تأكد أن لديك منتجات في Firebase
- امسح cache المتصفح وأعد المحاولة
