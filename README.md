## Tech Stack
- **HTML5 + CSS3 + Vanilla JavaScript ES6+**
- **Firebase** (Auth, Realtime Database, Storage, Analytics, Cloud Messaging)
- **PWA** (Service Worker, Web App Manifest)

## Setup
1. Clone the repository
2. Deploy to Firebase Hosting: `firebase deploy`
3. Configure admin access in Firebase Console (Custom Claims)
4. Add products via Admin Panel at `/pages/admin/`
